# Olympic Webscraper

Webscraper built with Typescript + axios + cheerio to get the current olympic medal count.

## Getting Started

1. Install packages

    ``` bash
    yarn 
    ```

2. Build files

    ```bash
    yarn build
    ```

3. Run script

    ```bash
    yarn start
    ```
